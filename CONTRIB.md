# Steps to Contribute

1. Install python3 (includes pip3)
Check the box at the bottom of the first Install Wizard screen to add python to PATH.
https://www.python.org/downloads/

1. Install pylint
`pip install pylint`

1. Install pygame
`pip install pygame`

1. Set up git config
`git config --global user.email "joannadelaporte@gmail.com"`
